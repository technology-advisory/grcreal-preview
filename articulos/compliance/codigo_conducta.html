El malentendido
«Tenemos código de conducta, firmado por todos»
Es la frase que todo auditor escucha. La empresa muestra un código de conducta de 50 páginas, con la firma de cada empleado en la última hoja. Y el auditor sabe, sin necesidad de preguntar, que nadie lo ha leído.

El código de conducta se ha convertido en un ritual burocrático: un documento que se entrega el primer día, que se firma sin leer, que se archiva en una carpeta y que nadie vuelve a consultar.

Y eso no es compliance. Eso es tener un papel.

La ley no exige tener un código de conducta. Exige tener un sistema de compliance que funcione. Y un código que nadie conoce no es un sistema que funciona.
Lo que exige la ley
El código como parte del sistema
Código Penal (art. 31 bis) El modelo de prevención penal debe incluir "medidas de vigilancia y control". Un código no es suficiente, pero es parte necesaria. Y debe ser conocido por todos.
Ley 2/2023 (whistleblowing) El sistema de gestión de información debe incluir "información clara y accesible sobre el canal". El código es el lugar natural para esa información.
Estándares ISO (37301, 37001) Exigen que el código esté documentado, comunicado, entendido y aplicado. No basta con tenerlo — hay que demostrar que vive.
Lo que falla
Los errores más comunes
El código es demasiado largo. 50 páginas de principios generales que nadie va a leer. Un código efectivo tiene 10-15 páginas como máximo.

El código es genérico. Descargado de internet, copiado de otra empresa. No habla de tu sector, de tus riesgos, de tu actividad. El empleado no se siente identificado.

No hay ejemplos reales. "No aceptar regalos" es vago. "No aceptar invitaciones de proveedores con contratos vivos por valor superior a 100€" es concreto. Los ejemplos importan.

No se comunica nunca más. Se entrega el primer día y nunca más se habla de él. La formación no existe. Las consultas no ocurren. El código está muerto.

No hay preguntas frecuentes. El empleado tiene una duda y no sabe dónde consultar. El código no incluye un "¿a quién pregunto si no entiendo algo?".

No se actualiza. La ley cambió hace dos años, tu código sigue igual. Los riesgos de tu sector evolucionaron, tu código no.

Y el error más grave: la dirección no lo cumple. El código prohíbe algo que el director hace. El código exige algo que el director ignora. El código no es para todos — es para los de abajo.
Cómo hacer uno que sirva
Las claves de un código vivo
1. Extensión justa. 10-15 páginas. Un ejecutivo lo lee en 20 minutos. Un empleado en media hora. Si es más largo, no se leerá.

2. Lenguaje claro. Nada de "en virtud de" o "a todos los efectos". Frases cortas, palabras sencillas. Un código no es un contrato.

3. Ejemplos reales. Para cada principio, un ejemplo de tu sector. "En nuestra empresa, no está permitido..." "Imagina que..."

4. Qué hacer si hay duda. Un apartado claro: "Si no sabes cómo actuar, pregunta a...". El compliance officer, el canal de denuncias, un correo específico.

5. Consecuencias del incumplimiento. No basta con "puede dar lugar a medidas disciplinarias". Especifica: amonestación, suspensión, despido, derivación a la vía penal.

6. Aprobación por la dirección. Firmado por el CEO o el consejo. Y con una declaración explícita: "este código me aplica a mí el primero".

7. Comunicación activa. No basta con enviarlo por email. Reunión explicativa, ejemplos en intranet, preguntas frecuentes, caso práctico en la formación anual.

8. Formación con caso práctico. No leer el código en formación. Usar un caso real (anonimizado) y preguntar "según nuestro código, ¿qué harías?".

9. Revisión anual. ¿Cambió la ley? ¿Hubo incidentes que el código no cubría? ¿Hay nuevas preguntas frecuentes? Revisar y actualizar.

10. La prueba del ascensor. Si preguntas a un empleado al azar "¿qué dice nuestro código sobre X?" y no sabe responder, tu código no sirve.
Código y cultura
El documento que refleja quién eres
Un código de conducta no es un documento legal. Es una declaración de valores. Es la respuesta a "¿qué tipo de empresa somos?".

Si tu código prohíbe el acoso pero toleras al jefe que acosa — tu código no vale nada.

Si tu código exige transparencia pero ocultas información — tu código es hipocresía.

Si tu código habla de integridad pero cierras los ojos ante prácticas cuestionables — tu código es un adorno.

El código de conducta no crea cultura. La cultura crea el código, o lo mata.

Por eso la dirección tiene que cumplirlo la primera. Por eso el compliance officer tiene que poder señalar incumplimientos del director general. Por eso el canal de denuncias tiene que ser real.

Un código que solo aplica a los de abajo no es un código de conducta. Es un manual de instrucciones para no ser despedido.
Autodiagnóstico
Preguntas para tu empresa
¿Cuántos empleados han leído el código completo?

¿Cuántas consultas ha recibido el compliance officer sobre el código en el último año? (cero es mala señal)

¿El código está visible en la intranet y se encuentra en menos de tres clics?

¿Los nuevos empleados reciben formación específica sobre el código, no solo un PDF por email?

¿Alguna vez se ha actualizado el código? ¿Cuándo fue la última vez?

¿La dirección habla del código en reuniones generales? ¿O solo existe en el departamento legal?

Si la respuesta a más de la mitad es "no", tu código no sirve.
